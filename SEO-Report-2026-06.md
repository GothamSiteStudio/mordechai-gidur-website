## 8. ממצאים טכניים קריטיים 🔴

זוהי הסיבה המרכזית שהאתר מתקשה לדרג, למרות שהתוכן והמבנה טובים:

### 8א. תגי Canonical מצביעים לדומיין שגוי (קריטי)
8 דפי השירות מכילים תג canonical שמצביע ל:
`https://www.mordechai-gidur.co.il/services/iskurit.html`
בעוד הדומיין האמיתי הוא:
`https://mordechaigidur.co.il/services/iskurit`

ההבדלים: `www` מיותר, **מקף** (`mordechai-gidur` במקום `mordechaigidur`), וסיומת `.html`. בפועל זהו **דומיין אחר לגמרי שלא קיים**. גוגל מתעלם מהתג ובוחר קנוניקל בעצמו (אישרנו זאת בבדיקת URL), אבל זה שולח אות סותר ומבזבז "אמון".

### 8ב. כפילות URL: `.html` מול URL נקי (קריטי)
- הקישורים הפנימיים וה-sitemap מפנים ל-`/services/iskurit.html`
- גוגל בחר לאינדקס את הגרסה הנקייה `/services/iskurit`
- שתי הגרסאות חיות ומחזירות אותו תוכן → גוגל רואה **כפילות** ומפצל את "כוח הדירוג".
- אישור: ב-GSC מופיעים גם `/service-areas/holon` וגם `/service-areas/holon.html`, וכן modiin, petah-tikva, ramat-gan, tree-protection בשתי הגרסאות.

### 8ג. סיטמאפ מדווח "0 באינדקס"
הסיטמאפ הוגש עם 34 כתובות בפורמט `.html`, אך גוגל מאנדקס את הגרסאות הנקיות → הסיטמאפ מדווח **34 הוגשו / 0 באינדקס**. בנוסף, הסיטמאפ לא נסרק מחדש מאז 04/01/2026 (כמעט חצי שנה).

### 8ד. רוב הדפים ללא תג canonical כלל
רק 11 דפים מכילים canonical. **דף הבית, כל 20 דפי אזורי השירות, services/index, accessibility ו-terms-of-use — ללא canonical כלל.** (חריג חיובי: privacy.html מפנה כראוי ל-privacy-policy.html.)

> 💡 **התקלות 8א–8ד מסבירות מדוע מילות ליבה תקועות בעמוד 3 למרות 234–525 הופעות.** תיקונן צפוי לשפר דירוגים תוך 4–8 שבועות, ללא עלות מדיה.

## 9. תוכנית קידום מפורטת

### שלב 0 — תיקונים טכניים (שבוע 1) 🔴 עדיפות עליונה, עלות אפס

1. **אחידות פורמט URL** — לבחור את הגרסה הנקייה (בלי `.html`), שגוגל כבר מעדיף:
   - לעדכן את כל הקישורים הפנימיים מ-`services/iskurit.html` ל-`/services/iskurit`.
   - להוסיף הפניית 301 מ-`.html` לגרסה הנקייה (תלוי באחסון — אם GitHub Pages, להגדיר בהתאם).
2. **לתקן את כל תגי ה-canonical** לדומיין הנכון, בפורמט נקי ועם הפניה-עצמית:
   `<link rel="canonical" href="https://mordechaigidur.co.il/services/iskurit">`
3. **להוסיף canonical לכל הדפים שחסר בהם** (דף הבית + 20 דפי אזורים + שאר).
4. **לעדכן את ה-sitemap.xml** לכתובות נקיות בלבד ולעדכן `lastmod`, ואז **להגיש מחדש** ב-Search Console.
5. לאחר התיקון — להשתמש בכלי "בדיקת כתובת URL" ולבקש אינדוקס מחדש לדף הבית, איסכורית, רשת, שילוט.

> 💡 אני יכול לבצע את כל שלב 0 בקוד עכשיו (תיקון canonical-ים, sitemap, קישורים פנימיים) — רק תאשר.

### שלב 1 — אופטימיזציית On-Page למילות הליבה (שבועות 2–4)

מיקוד בדפים עם ביקוש גבוה ומיקום נמוך:

1. **דף איסכורית** (1,399 הופעות, מיקום 33) — היעד מס' 1:
   - לחזק את הכותרת (Title) וה-H1 סביב "גדר איסכורית / גידור איסכורית / גדר פח איסכורית".
   - להרחיב תוכן: מה זו גדר איסכורית, יתרונות, מחיר למטר (יש ביקוש ל"גדר איסכורית מחיר"), גלריית פרויקטים, שאלות נפוצות (FAQ Schema).
   - קישורים פנימיים מדף הבית ומדפי האזורים אל דף האיסכורית עם עוגן "גדר איסכורית".
2. **דף הבית** — לחזק "קבלן גידור" (כבר במיקום 9.6) ו"גידור אתרי בניה" בכותרת, בתיאור וב-H1, להוסיף קריאה לפעולה ברורה ומספר טלפון בולט.
3. **דף רשת** (139 הופעות, 0 קליקים, מיקום 38) ו**דף שילוט** (238 הופעות, מיקום 54) — להעשיר תוכן ולמקד ב"גדר רשת זמנית" / "שילוט לאתרי בניה".
4. **שיפור CTR** — לכתוב Title ו-Meta Description מושכים עם USP (ביטוח קבלני, עמידה בתקנים, הקמה מהירה, טלפון) לכל הדפים המובילים.
5. **Schema** — להוסיף `LocalBusiness` בדף הבית ו-`Service` בדפי השירות; כבר זוהה Breadcrumbs — להרחיב.

### שלב 2 — SEO מקומי + Google Business Profile (חודש 2)

הגידור הוא עסק מקומי מובהק → SEO מקומי הוא ה-ROI הגבוה ביותר:
1. **לפתוח/לאמת Google Business Profile** (אם אין) — קטגוריה "קבלן גידור", אזורי שירות, תמונות פרויקטים, טלפון, שעות. זה לבדו פותח דלת ל-Map Pack.
2. לעודד **ביקורות לקוחות** (חיפושי מותג ואמון).
3. לחזק את דפי האזורים שכבר מדורגים בעמוד 1 (חדרה 6.4, ראשון 5.6, באר שבע 9.3, בית שמש 4.7) — להוסיף בהם תוכן ייחודי, פרויקטים מקומיים, ולקשר אליהם מדף הבית.
4. **ציטוטים מקומיים (NAP)** — שם/כתובת/טלפון אחידים באינדקסים ישראליים (דפי זהב, b144, אתרי קבלנים).

### שלב 3 — תוכן והרחבת ביקוש (חודשים 2–3)

1. **דפי מחיר/מדריך** — יש ביקוש ל"גדר איסכורית מחיר", "גידור אתרי בניה מחיר" → ליצור עמוד "מחירון גידור אתרי בניה 2026" עם טבלת מחירים מנחה.
2. **בלוג/מאמרים**: "כמה עולה גידור אתר בניה?", "גדר איסכורית מול גדר פאנל", "תקנים לגידור אתרי בניה", "גידור זמני לאירועים".
3. **שימוש חוזר בתוכן** — להמיר מאמרים לפוסטים ברשתות ול-FAQ.

### שלב 4 — סמכות וקישורים (לאורך זמן)

1. קישורים מ-**אתרי תעשייה ובנייה**, איגודי קבלנים, ספקים.
2. שיתופי פעולה עם קבלני בניין (מקור הפניות טבעי).
3. נוכחות ב-LinkedIn/פייסבוק עם פרויקטים — תומך במותג ובקישורים.
